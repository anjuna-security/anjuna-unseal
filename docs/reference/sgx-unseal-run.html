Anjuna Vault Unseal¶

NAME¶

anjuna-vault-unseal: Securely Unseal a Vault instance by running the unsealing tool in the Anjuna Vault-Unseal tool.

SYNOPSIS¶

$ anjuna-vault-unseal

DESCRIPTION¶

The Anjuna Vault Unseal utility uses the encrypted configuration file created by the Vault Unseal Configuration tool to perform the Vault Unsealing procedure.

This tool MUST be run from the directory where the Vault Unseal Configuration was created.

OPTIONS¶

--help          : Print the help summary and exit

EXIT STATUS¶

  • 0 on success
  • 1 if an error occurred during the Vault Unsealing process

EXAMPLE¶

$ anjuna-vault-unseal
Found Vault Unseal configuration => unsealing now...
2019/03/05 11:01:23 - Unsealing Vault https://vault.anjuna.test:9980 using token 1/2
2019/03/05 11:01:23 - Unsealing Vault https://vault.anjuna.test:9980 using token 2/2
2019/03/05 11:01:23 Vault is now unsealed
