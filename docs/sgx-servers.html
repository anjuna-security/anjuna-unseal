Getting access to Intel® SGX¶

Software Guard Extensions (SGX) are available on Intel® processors starting with Skylake architecture and on (Kaby Lake, Cannon Lake, Coffee Lake).

For production deployment and performance benchmarking, SGX is available on Intel® Xeon E3 (v5 and on) and Intel® Xeon E 2100 series servers.

Note

List of Xeon CPU models with SGX supports:

  • Xeon E3 v5: 1575M, 1545M, 1535M, 1515M, 1505M, 1505L, 1280, 1275, 1270, 1268L, 1260L, 1245, 1240L, 1240, 1235L, 1230, 1225, 1220
  • Xeon E3 v6: all models
  • Xeon E: all 2100-series models

For development and testing purposes, a development Intel® NUC can be sufficient.

Note

List of NUC models with SGX support:

  • 6-th gen: NUC6i3SYK, NUC6i3SYH, NUC6i5SYK, NUC6i5SYH, NUC6i7KYK
  • 7-th gen: NUC7i3BNK, NUC7i3BNHX1, NUC7i3BNH, NUC7i5BNK, NUC7i5BNH, NUC7i5BNHX1, NUC7i3DNKE, NUC7i3DNHE, NUC7i5DNKE, NUC7i5DNHE, NUC7i7DNKE, NUC7i7DNHE, NUC7i7BNH, NUC7i7BNHX1, NUC7PJYH, NUC7CJYH
  • 8-th gen: NUC8i3BEK, NUC8i3BEH, NUC8i5BEK, NUC8i5BEH, NUC8i7BEH, NUC8i7HNK, NUC8i7HVK, NUC8i3CYSM, NUC8i3CYSN

Cloud Providers Supporting Intel® SGX¶

  • Microsoft Azure - available as part of Azure Confidential Computing on DC2 instances.
  • Packet.com – on c1.xmall.x86 instances.
  • IBM Cloud
  • Vultr – on bare metal instances.
  • Alibaba Cloud – on ECS bare metal instances.
  • OVH Cloud – on INF 1 instances.
  • Leaseweb
